# DentalPro Mobile

Expo React Native app for DentalPro clinic management.

## Scripts
- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`

## Run
1. Install dependencies with `npm install`
2. Ensure backend API is running
3. Update `src/api/client.js` URL if needed
4. Run `npm run start`
